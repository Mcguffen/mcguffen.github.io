运行 node.js

不要双击 Node.js 快捷方式

如何运行 Node.js：

在cmd 里运行 node

按 ctrl + D 退出

node 接绝对路径

在一个目录下新建 1.js 文件并用 vscode 打开它

然后回到命令行用绝对路径去执行它

node 接相对路径

直接在目录下执行 1.js 文件

不接文件名一样可以执行

node -e 接字符串

不用文件直接执行代码
