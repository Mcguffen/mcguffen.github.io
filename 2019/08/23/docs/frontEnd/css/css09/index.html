IFC布局规则：

  1. 框会从包含块的顶部开始，一个接一个地水平摆放。
  2. 摆放这些框时，它们在水平方向的 内外边距+边框 所占用的空间都会被考虑； 在垂直方向上，这些框可能会以不同形式来对齐： 水平的margin、padding、border有效，垂直无效。不能指定宽高;
  3. 行框的宽度是 由包含块和存在的浮动来决定; 行框的高度 由行高来决定
