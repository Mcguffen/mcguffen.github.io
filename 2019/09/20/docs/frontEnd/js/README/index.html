http-server 的使用方法

  1. 安装 node
  2. npm i -g http-server
  3. 进入项目目录
  4. 运行 http-server . -c-1 （一个字都不能错）
  5. 用浏览器打开 http://127.0.0.1:xxxx （xxxx 是对应的端口号）
  6. 如果发现页面不存在，就在地址栏后面加 /index.html ，变成 http://127.0.0.1:xxxx/index.html
  7. （可选）打开开发者工具的 Network，勾选 Disable Cache
